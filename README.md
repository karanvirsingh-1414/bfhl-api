# REST API

A Spring Boot REST API for processing various mathematical and logical operations.

## Features
- Fibonacci Sequence Generation
- Prime Number Filtering
- LCM Calculation
- HCF (GCD) Calculation
- AI-Powered Q&A via Google Gemini

## Deployment
This project is configured for deployment on platforms like Railway.

### Environment Variables
- `PORT`: Server port (default 8080)
- `GEMINI_API_KEY`: Google Gemini API Key
